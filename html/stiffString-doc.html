Integrated Stiff String Module:

@[name] stiffString  [nb_masses][inertia] [stiffness] [2nd order stiffness] [damping] [init_position]

This module is an integrated stiff string (each mass being connected to its second order neighbors).

Physical Parameters: Number of masses, inertia per mass, stiffness, 2nd order stiffness and damping for springs, initial position.
